@ice/dike-core-js - v0.2.3

dike-core-js

Api Wrapper javascript

Requisiti
  • Nodejs >= 12 (su windows è necessaria la versione a 32bit )
  • yarn (consigliato)

Utilizzo

Il modulo è ospitato sulla repository Artifactory di Inforcert Per poterlo utilizzare è necessario prima aggiungere la repository:

Aggiunta e autenticazione della repository Artifactory
  1. Loggarsi (tramite vpn) su Artifcatory
  2. Cercare tra le repo npm-ice-local e cliccarla
  3. Inserire la propria password (la stessa usata per il login) in alto a destra e autenticarsi
  4. Copiare le stringhe di autenticazione più in basso e incollarle nel file ~/.npmrc nella propria cartella user (su Windows %USERPROFILE%/.npmrc):
  5. sostiuire con ice, il contenuto del file .npmrc dovrà essere simile a questo:
    @ice:registry=https://integrazione.mgmt.infocert.it/artifactory/api/npm/npm-ice-local/
//integrazione.mgmt.infocert.it/artifactory/api/npm/npm-ice-local/:_password=###
//integrazione.mgmt.infocert.it/artifactory/api/npm/npm-ice-local/:username=yye9326
//integrazione.mgmt.infocert.it/artifactory/api/npm/npm-ice-local/:email=simone.lai.ext@infocert.it
//integrazione.mgmt.infocert.it/artifactory/api/npm/npm-ice-local/:always-auth=true
  6. Adesso è possibile scaricare i moduli con scope @ice dalla repository inforcert.

Installazione libreria nel progetto

Aggiungere il modulo alle dipendenze del progetto come un normale modulo npm:

yarn add @ice/dike-core-js

oppure

npm i @ice/dike-core-js

La prima volta è consigliabile eliminare la cartella node_modules e i file .lock per fare un'installazione pulita.

Utilizzo Api Wrapper

Importare il modulo e instanziare uno dei metodi ad esmpio:

//importare la libreria
import DIKE_CORE_API from "dike-core-js"

//Instanziare la classe
const DIKE_CORE = new DIKE_CORE_API();

//invocare i metodi
const version = DIKE_CORE.getVersion();
console.log("version", version);

const devices = DIKE_CORE.getDevices();
console.log("devices", devices)

La libreria utilizza comandi e moduli nativi, quindi per utilizzare la libreria in Electron è preferibile eseguirla nel Main thread (e non nel render)
